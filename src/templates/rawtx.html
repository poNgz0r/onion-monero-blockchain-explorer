Transaction Pusher

Paste here either a hex string of raw transaction (the tx_blob response in the wallet RPC, or the raw_FURY_tx file saved by the wallet CLI with --do-not-relay option specified), or base64 encoded, unsigned or signed transaction data (In Linux, can get the raw tx data: cat raw_FURY_tx | xclip -selection clipboard) (In Windows, can get the raw tx data: certutil.exe -encode -f raw_FURY_tx encoded.txt & type "encoded.txt" | clip) Note: data is sent to the server, as the calculations are done on the server side Note: "check" does not work for the hex string of raw transaction
